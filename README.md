# PDF Editor

PDFファイルの「ページの並び替え」「結合」「削除」ができるWebツールです。直感的な操作でPDFの再構成を行うことができます。

## 主な機能

- **ドラッグ&ドロップ操作**: ページの順番をドラッグで自由に入れ替えられます。
- **PDFの結合**: 複数のPDFをまとめてアップロードし、一つのファイルとして保存できます。
- **ページ削除**: 不要なページをサムネイル上のボタンから個別に削除できます。
- **プレビュー（拡大表示）**: 各ページをクリックして内容を確認できます。
- **Undo（元に戻す）**: 並び替えや削除の操作を1ステップずつ元に戻せます。
- **自動クリーンアップ**: 作業が終わってブラウザを閉じたり、一定時間が経過すると、サーバー上の一時ファイルは自動的に削除されます。

## ローカルでの実行方法

### 1. 依存ライブラリのインストール
Pythonがインストールされた環境で、以下のコマンドを実行して必要なライブラリをインストールします。

```bash
pip install -r requirements.txt
```

### 2. アプリの起動
macOS でのポート競合を避けるため、デフォルトで **ポート 8080** を使用する設定になっています。

```bash
python app.py
```

### 3. ブラウザでアクセス
起動後、ブラウザで以下のURLを開いてください。
[http://localhost:8080](http://localhost:8080)

## 使い方

1. **アップロード**: 画面中央のエリアにPDFファイルをドロップするか、ボタンからファイルを選択します。
2. **編集**: 
   - 表示されたサムネイルをドラッグして順番を入れ替えます。
   - サムネイルをクリックすると、そのページを拡大して確認できます。
   - 「＋ Add Files」ボタンから別のPDFを追加することも可能です。
3. **保存**: 「Save PDF」ボタンを押すと、現在の順序で結合された新しいPDFがダウンロードされます。
4. **クリア**: 「Clear All」ボタンを押すと、現在の作業内容とファイルがサーバーから完全に削除されます。

## 技術要素
- **Backend**: Python (Flask, PyMuPDF, pypdf)
- **Frontend**: JavaScript, Sortable.js, CSS
