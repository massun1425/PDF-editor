# PDF Editor

PDFファイルの「ページの並び替え」「結合」「削除」ができるWebツールです。直感的な操作でPDFの再構成を行うことができます。

## 主な機能

- **ページの回転**: サムネイル上の「↻」ボタンから、ページごとに90度ずつ回転させ、正しい向きに修正できます。
- **画像ファイルの取り込み**: PDFだけでなく、JPGやPNG画像をアップロードしてPDFのページとして統合できます（画質の劣化なし）。
- **パスワード保護**: 保存時にパスワードを設定して、PDFを暗号化できます。
- **直感的な操作**: ページの順番をドラッグ&ドロップで自由に入れ替え、不要なページは個別に削除できます。
- **プレビュー機能**: 各ページをクリックして拡大表示し、細かい内容を確認できます。
- **Undo（元に戻す）**: 並び替えや削除の操作を1ステップずつ元に戻せます。
- **セキュアな設計**: ユーザーごとにセッションが隔離されており、作業終了時や一定時間経過後にサーバー内のファイルは自動的にクリーンアップされます。

## ローカルでの実行方法

### 1. 依存ライブラリのインストール
Pythonがインストールされた環境で、以下のコマンドを実行して必要なライブラリをインストールします。

```bash
pip install -r requirements.txt
```

### 2. アプリの起動
macOS でのポート競合を避けるため、デフォルトで **ポート 8080** を使用する設定になっています。

```bash
python app.py
```

### 3. ブラウザでアクセス
起動後、ブラウザで以下のURLを開いてください。
[http://localhost:8080](http://localhost:8080)

## 使い方

1. **アップロード**: 画面中央のエリアにPDF、または画像ファイル（JPG/PNG）をドロップするか、ボタンからファイルを選択します。
2. **編集**: 
   - 表示されたサムネイルをドラッグして順番を入れ替えます。
   - 向きが違うページは「↻」ボタンで回転させることができます。
   - サムネイルをクリックすると、拡大プレビューで内容を確認できます。
   - 「＋ Add Files」ボタンから別のファイルを追加することも可能です。
3. **保存**: 必要に応じてパスワードを入力し、「Save PDF」ボタンを押すと、現在の順序で結合された新しいPDF (`edited.pdf`) がダウンロードされます。
4. **クリア**: 画面のリセットや「Clear All」ボタンにより、現在の作業内容とファイルがサーバーから完全に削除されます。

## 技術要素
- **Backend**: Python (Flask, PyMuPDF, pypdf)
- **Frontend**: JavaScript, Sortable.js, CSS
